Purpose

The purpose of this project was to create a pulse oximeter using low cost
parts.  At the current time, this can only be used to measure heart beat.  The
parts include an AVR (uses commonly available Arduino hardware), a programable
light sensor chip, and LEDs of proper frequencies.

Information

For more information, please see the project blog at:
http://tinkerish.com/blog/?p=181

* main.c, Makefile - main project
* tsl230.c - TSL230 light sensor "driver"
* libarduino.c, libaruduin.h - abstraction layer for arduino hardware
* graphy.py - python and QT visualization tool
